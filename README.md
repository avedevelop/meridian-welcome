# Meridian Welcome Vaults

Stock welcome vaults for Meridian.

## Layout

- `macos/en` — macOS, English
- `macos/nb` — macOS, Norwegian bokmål
- `macos/ru` — macOS, Russian
- `windows/en` — Windows, English
- `windows/nb` — Windows, Norwegian bokmål
- `windows/ru` — Windows, Russian

Meridian downloads this repository as a ZIP on first launch and copies only the
matching platform/language vault into the user's Documents folder.
